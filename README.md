# bodyct-luna22-ismi-baseline
Repo for training the baseline algorithm for LUNA22-ismi challenge.

## Training
To train the algorithm, just execute the following script:
```bash
python train.py
```

## Changes
- Created a 2d and a 3d CNN partly based on [1]
- Added rotation to the preprocessing pipeline
- Outputs used later on to do ensemble learning

## References
[1] Ping An Technology (Shenzhen), “3DCNN for False Positive Reduction
in Lung Nodule Detection”, unpublished, China, 2017
